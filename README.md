Vim IDE like setup for Go

Prerequisits
============
node > v10.16.0
yarn > 1.16.0

Install nvim
https://github.com/neovim/neovim/wiki/Installing-Neovim

Install plug
https://github.com/junegunn/vim-plug

Install gopls
go get -u golang.org/x/tools/gopls

cd ~/.config
git clone https://github.com/shrkamat/nvim
nvim +PlugInstall
